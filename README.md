•Description
This project is a web-based geographic information system (GIS) mapping application focused on the city of Batumi. It visualizes spatial data and provides an interactive experience for users to explore the city's features.

•Features

Interactive map interface.

Spatial data visualization using GIS technologies.

User-friendly navigation for exploring Batumi's locations.

•Technologies Used
GIS (Geographic Information Systems)

•How to Run
Simply open the index.html file in any modern web browser to view the map.
